# Zodiac Calendar

A custom 13-month calendar web application featuring unique zodiac-named months and multi-language support.

## Features

- **Custom 13-Month System**: A unique calendar logic with 13 months, including a special "Sol" month.
- **Zodiac Month Names**: Months are named after zodiac signs (e.g., Aquarius, Pisces, Aries).
- **Multi-Language Support**: Supports 13 languages including English, Spanish, French, German, and more.
- **Interactive Interface**:
  - Highlights the current date.
  - Easy navigation between months.
  - Responsive design for various devices.
- **Date Converter**: A tool to convert between Gregorian dates and the custom Zodiac calendar.
- **Firebase Integration**:
  - User authentication (Email/Google).
  - Cloud storage for date-specific notes and tags (planned/in-progress).

## Setup

1.  Clone the repository.
2.  Open `index.html` in your web browser to view the calendar.
3.  For full functionality (including Firebase features), you may need to serve the project using a local server (e.g., Live Server in VS Code or `python -m http.server`).

## Technologies

- HTML5
- CSS3 (Custom properties, Flexbox/Grid)
- JavaScript (ES6+)
- Firebase (Auth, Firestore)
